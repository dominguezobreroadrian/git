Examen Git y Github

Alumn@: A.D.O

  1. ¿Que es un sistema de control de versiones?

    Un sistema de control de versiones (VCS) es una herramienta que permite registrar y gestionar los cambios realizados en archivos a lo largo del tiempo. Facilita trabajar en equipo, volver a versiones anteriores y mantener un historial de modificaciones.

  2. ¿Cual es la diferencia entre git y github?

    Git - Es un sistema de control de versiones que se instala en tu ordenador para manejar versiones de código. Github - Es una plataforma en la nube que permite alojar repositorios Git, colaborar con otros desarrolladores y gestionar proyectos.

  3. ¿Cómo creo una rama en git?

    git branch nombre-rama

  4. ¿Cómo clono un repositorio?

    git clone https://github.com/usuario/repositorio.git

  5. ¿Cómo puedo bajar los ultimos cambios de un repositorio?

    git pull origin main

  6. ¿Cómo puedo gener un tag?

    git tag nombre-tag

  7. Cómo hago una mezcla de ramas?

    Primero cámbiate a la rama destino: git checkout main Luego fusiona la otra rama: git merge nombre-rama

  8. ¿Cómo puedo moverme a un commit especifico?

    git checkout hash-del-commit

  9. ¿Cómo puedo borrar una rama?

    git branch -d nombre-rama

  10. ¿Cómo puedo borrar un tag?

    git tag -d nombre-tag

  11. ¿Cómo puedo ver los cambios que estan por enviarse al repositorio?

    git log origin/main..HEAD

  12. ¿Para que sirve el comando git remote -v?

    Muestra las URLs de los repositorios remotos configurados en tu proyecto, tanto para fetch como para push.

  13. ¿Cómo le digo a git cual es mi email y correo?

    git config --global user.name "Tu Nombre" git config --global user.email "tu@email.com"

  14. ¿Cuales son los pasos para conectar git con github?

    Crear un repositorio en GitHub. Inicializar repositorio local: git init Agregar archivos y hacer commit: git add . git commit -m "Primer commit" Agregar el repositorio remoto: git remote add origin https://github.com/usuario/repositorio.git Enviar los cambios: git push -u origin main

  15. ¿Cómo puedo crear un alias en git?

    Crear un alias global: git config --global alias.co checkout git config --global alias.br branch git config --global alias.cm commit git config --global alias.st status
